- Converting ui file that designed by QtDesigner to python file

pyuic5 -x windows/MainWindow.ui -o windows/MainWindow.py
pyuic5 -x windows/ConnectingWindow.ui -o windows/ConnectingWindow.py
pyuic5 -x windows/WaypointWindow.ui -o windows/WaypointWindow.py
pyuic5 -x windows/WaypointsSettingWindow.ui -o windows/WaypointsSettingWindow.py

- If they're not working use 

python3 -m PyQt5.uic.pyuic -x windows/MainWindow.ui -o windows/MainWindow.py

- Installing PyQt5

pip install PyQt5 / sudo apt install python3-pyqt5
pip install PyQtWebEngine / sudo apt install python3-pyqt5.qtwebengine

- If there is an error use

export PATH="/opt/homebrew/opt/qt5/bin:$PATH"

- Installing QtDesigner

sudo apt-get install qttools5-dev-tools
sudo apt-get install qttools5-dev
